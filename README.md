## Data Quality Platform:

Improve your datasets before training machine learning models.
A platform that analyzes data quality, identifies potential issues, and provides actionable insights to help build more reliable and efficient ML pipelines.
Inspired by real academic research on the relationship between data quality, computational resources, and machine learning performance.



## Why this project:

This project was born from a university research study that investigated how improving data quality affects computational resources and predictive performance in machine learning.

While conducting the research, one question emerged:
How can we make these data quality practices accessible through software instead of keeping them only in academic papers?
Data Quality Platform is my answer to that question.



## The Problem
Machine learning models are often trained using datasets that contain missing values, duplicates, inconsistent formats, noisy records and outliers.

Poor data quality can reduce model performance, increase training iterations and lead to unnecessary computational costs.

Most existing tools focus on detecting data quality issues.

This project explores how data quality can become part of a better machine learning workflow.




## Why Data Quality Platform?

Unlike traditional data profiling tools, this project is inspired by experimental research on how data quality impacts machine learning.

The goal is not only to detect problems inside datasets, but also to provide insights about how improving data quality may influence future ML workflows.



## Features

- Dataset profiling
- Missing value detection
- Duplicate detection
- Outlier detection
- Data quality score
- Cleaning recommendations
- Estimated impact based on research
- Exportable reports



## Academic Background

This project is inspired by a university research study that evaluated how data quality influences machine learning performance and computational efficiency.

The software is not an implementation of the research pipeline.

Instead, it translates part of those findings into a practical tool for machine learning practitioners.
